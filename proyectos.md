# Proyectos más importantes

## Sistema web para gestión de documentos escolares
Desarrollo de un prototipo de aplicación web orientada a digitalizar la carga y solicitud de documentos académicos, mejorando la organización y reduciendo tiempos de atención.

## Sistema de inventario para negocio local
Diseño e implementación de un sistema básico para registrar productos, controlar existencias y generar consultas rápidas sobre el inventario.

## Portafolio web personal
Creación de un sitio web responsivo para presentar perfil profesional, habilidades, proyectos y datos de contacto.

## Base de datos para cafetería
Modelado e implementación de una base de datos objeto-relacional para administrar productos, empleados, sucursales y ventas.

## Aplicación de tareas o recordatorios
Desarrollo de una aplicación sencilla para registrar, editar y eliminar tareas, con una interfaz intuitiva para el usuario.
